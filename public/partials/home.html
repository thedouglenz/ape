What is Ape?

AngularJS

On the frontend, there's AngularJS. One of the most powerful tools for creating dynamic views for your web application. Create fast frontend logic that connects easily to your backend APIs. The facilities to do so are already built in to APE.

PostgreSQL

On the backend, the champion of relational databases. The last thing we need is to muck around with document oriented database systems when we're dealing with important data! Persistent storage of your valuable data is powered by Sequelize and the best in relational databases, PostgreSQL.

Express

In between -- the master of Node.js web frameworks. Build a powerful RESTful API with ease using the best solution for Node.js app server-sides available.
